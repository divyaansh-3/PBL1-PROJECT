# AI-Based ICU Mortality Prediction

## 📌 Project Overview
This project predicts the probability of ICU patient mortality using machine learning models based on patient vitals, lab values, and clinical scores.

---

## 🚀 Features
- Data preprocessing and cleaning
- Feature engineering
- Logistic Regression and Random Forest models
- Evaluation using Accuracy, Recall, Precision, and AUC
- ROC Curve visualization
- Risk stratification (Low / Medium / High risk)

---

## 📊 Dataset
- ICU patient dataset (dummy / MIMIC-based)
- Includes vitals, lab results, and severity scores

---

## 🧠 Models Used
- Logistic Regression
- Random Forest Classifier

---

## 📈 Evaluation Metrics
- Accuracy
- Recall (important for critical patients)
- Precision
- ROC-AUC Score

---

## ⚠️ Limitations
- Dataset imbalance
- Limited features (first 24 hours only)
- Not a real-time system

---

## 🔮 Future Scope
- Use Deep Learning models
- Real-time ICU monitoring
- Integration with hospital systems
